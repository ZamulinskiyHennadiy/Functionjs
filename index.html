Види функції

  • Оголошення функції

    function myFunction() { }

  • Функціональний вираз

    const myFunction = function() { };

  • Стрілкова функція

    const myFunction = () => { };

  • Анонімна функція

    setTimeout(function() { }, 1000);

  • Самовиконувана функція

    (function() { })();

  • Метод об'єкта: Функція, яка є методом об'єкта.

    const obj = { myMethod: function() { } };

  • Генератор функцій: Функція, що може зупинятись і продовжувати виконання.

    function* myGenerator() { yield 1; yield 2; }
